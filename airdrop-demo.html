Jewel ICU Airdrop Demo

This is a demonstration of the airdrop functionality

Jewel ICU

How the Airdrop Works:

  1. Click "Simulate Wallet Connection" to see the connected state
  2. The "Claim Airdrop" button will appear next to the wallet info
  3. Click "Claim Airdrop" to see the airdrop process
  4. The button will show a loading state
  5. After 2 seconds, you'll see a success message

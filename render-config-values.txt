4. HOW TO GENERATE REQUIRED VALUES
-----------------------------------

A. Generate ADMIN_API_KEY:
Run this command in your terminal:
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

Example output: 7f3a8b2c9d4e5f6a7b8c9d0e1f2a3b4c5d6e7f8a9b0c1d2e3f4a5b6c7d8e9f0a

B. Get AIRDROP_PRIVATE_KEY:
If you don't have a wallet yet, create one:
cd airdrop-backend
node create-wallet.js

Then convert to base64:
node convert-key.js

C. Get RECAPTCHA_SECRET_KEY:
1. Go to https://www.google.com/recaptcha/admin
2. Click "+" to create new site
3. Label: Jewel ICU Airdrop
4. reCAPTCHA type: reCAPTCHA v3
5. Domains: Add your frontend domain
6. Accept terms and Submit
7. Copy the SECRET KEY

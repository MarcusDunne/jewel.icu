Jewel ICU Airdrop Test Page

Test Instructions

  1. Open the main index.html page in your browser
  2. Connect your Solflare wallet
  3. Once connected, you should see the "Claim Airdrop" button appear
  4. Click the "Claim Airdrop" button
  5. The button should show a loading state
  6. After ~2 seconds, you should see a success toast message

Production Considerations

To make this airdrop functionality production-ready, you would need:

  • Server-side component: A backend service that holds the airdrop wallet's private key
  • Rate limiting: Prevent users from claiming multiple airdrops
  • Database: Track which wallets have already claimed their airdrop
  • Verification: Implement eligibility criteria (e.g., wallet age, balance requirements)
  • Security: Protect against bot attacks and Sybil attacks
